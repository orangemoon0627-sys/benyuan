# 本源 Beta 测试者说明

欢迎参与本源 Beta 测试。本次测试重点是确认你是否能稳定完成一次精神探索，以及结果页是否有共鸣、清晰、值得保存。

## 测试链接

请从 Web staging 进入：

http://120.26.126.88/collect

如果你收到的是 TestFlight 版本，也请从 App 打开后按同样流程测试。

## 测试任务

1. 从 `/collect` 开始，选择一套测试入口并提交内容。
2. 等待进入 `/processing/benyuan`，观察生成过程是否清楚、有无卡住。
3. 进入 `/theater`，完成剧场选择。
4. 到 `/constellation` 查看精神星图结果页。
5. 在结果页底部尝试 `分享`、`保存`、`重新探索`。
6. 完成后填写反馈表。

## 重点观察

- 哪一步最顺，哪一步最困惑。
- 结果页中的哪一句最有共鸣。
- 哪个结果让你觉得不准确、过度解读或不够具体。
- 上传、等待、返回、刷新、分享、保存是否稳定。
- 如果使用 iPhone，请记录 iPhone 型号和 iOS 版本。

## 测试边界

本源 Beta 结果是一种理解性镜像，不是医疗、心理诊断或治疗建议。如果过程中感到不适，可以随时退出，不需要完成全部流程。
