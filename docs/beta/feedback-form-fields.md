# 本源 Beta 反馈表字段

建议用飞书表单、腾讯问卷、Notion 或 Google Form 建一份表。字段尽量短，测试者能在 3 分钟内填完。

## 基础信息

- 测试日期
- 测试入口：Web staging / iOS TestFlight / 其他
- iPhone 型号
- iOS 版本
- 网络环境：Wi-Fi / 5G / 其他

## 流程反馈

- 是否完整跑完：是 / 否
- 卡住的步骤：`/collect` / `/processing/benyuan` / `/theater` / `/constellation` / 分享保存 / 其他
- 卡住时发生了什么
- 等待时间感受：太短 / 合适 / 偏长 / 过长

## 内容反馈

- 最有共鸣的内容
- 不准确的结果
- 看不懂或太抽象的句子
- 你希望结果页多给什么：更具体行动 / 更短总结 / 更多解释 / 更多推荐 / 其他

## 体验反馈

- 视觉感受：喜欢 / 一般 / 不喜欢
- 操作是否顺手：顺手 / 有一点卡 / 很卡
- 分享是否成功
- 保存是否成功
- 重新探索是否清楚

## 复测意愿

- 是否愿意复测
- 是否愿意加入下一轮 TestFlight
- 可联系邮箱或微信
- 其他建议
